## ✨ 서비스 소개 ✨
기존 SEMES 기업은 반도체 포토 공정에서 검사 과정은 검사 모듈에 개별적으로 접근하여 결과를 확인했다.
따라서 개별적으로 접근하는 불편함을 없애고자 검사 결과를 통합적으로 모니터링하는 시스템을 구축하고자 함

### 1. 웨이퍼 목록 조회
